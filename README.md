# design-patterns-in-deno

Design Patterns in Deno (TypeScript)

## Installation

Install Deno

```bash
https://github.com/denoland/deno
```

## Configuration

- change `deno.path` in `.vscode/settings.json`

## Test Design Patterns

in project root folder run:

```bash
  deno test
```
